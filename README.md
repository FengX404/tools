| 工具 | 说明 |

详细使用说明请查看 [TOOLS.md](TOOLS.md)

## 使用方式

每个工具都是独立的脚本，可以直接运行：

```bash
python <工具名>.py [参数]
```

或添加执行权限后直接运行：

```bash
chmod +x <工具名>.py
./<工具名>.py [参数]
```
